## 객체 리터럴 📝

### 객체란?

- javascript는 객체 기반의 프로그래밍 언어이며, 자바스크립트를 구성하는 거의 모든것이 객체!!!

### 객체 리터럴에 의한 객체 생성

```
let obj = {};
// 끝!

```

### 프로퍼티 동적 생성

```
let obj = {};
obj['key'] = '12345';
obj.age = 20;

```

### ES6에서 추가된 객체 리터럴의 확장기능

- 프로퍼티 축약 표현

```
let x = 1, y = 2;
let obj = { x, y };
// { x : 1, y : 2}

```

- 계산된 프로퍼티 이름
- 메서드 축약 표현

```
const obj = {
    name : "songsong",
    sayHi() {
        console.log("Hi!");
    }
}

```
